# Test Guide

This directory contains unit tests for the `sum` functions implemented in the project.

---

## Structure

```
tests/
├── jest.config.js     # Jest configuration
├── package.json       # Test dependencies & scripts
├── sum.test.ts        # Test cases for sum functions
└── node_modules/      # Dependencies
```

---

## Getting Started

### 1. Install dependencies

```bash
npm install
```

---

### 2. Run tests

```bash
npx jest
```

---

## Test Coverage

The test file `sum.test.ts` covers:

* Iterative implementation
* Formula-based implementation
* Recursive implementation
